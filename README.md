# Todo Kanban
Simple todo list similar to Trello.


## Live Project
https://todo-kanban-five.vercel.app


## How to use
**Install NodeJs packages**

    $ yarn install

**Run project on local**

    $ yarn dev

**Run database**

    $ docker-compose up -d

**Config env variables**

Rename  __.env.example__  to  __.env__



**Seed data base**
````
http://localhost:3000/api/seed
````
